# 18_Swagger_Employee_API_Explorer

A simple ASP.NET Core Web API project that demonstrates Employee CRUD operations with Swagger API Documentation, EF Core, SQL Server, Data Annotations Validation, and Fluent API Configuration.

## Technologies Used

- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Swagger
- Dependency Injection
- Data Annotations
- Fluent API

## Features

### Employee API

- GET All Employees
- GET Employee By Id
- POST Employee
- PUT Employee
- DELETE Employee

### Validation

- Required Field Validation
- Email Format Validation
- Phone Number Validation
- Salary Range Validation

### Database Configuration

- Entity Framework Core
- SQL Server Database
- Fluent API Configuration
- Decimal Precision Configuration

### Swagger Features

- API Documentation
- API Testing Interface
- Request and Response Models
- Endpoint Exploration

## Employee Model

| Property | Type |
|-----------|------|
| Id | int |
| FirstName | string |
| LastName | string |
| Email | string |
| PhoneNumber | string |
| Department | string |
| Salary | decimal |

## API Endpoints

### Get All Employees
GET /api/Employee

### Get Employee By Id
GET /api/Employee/{id}

### Create Employee
POST /api/Employee

### Update Employee
PUT /api/Employee/{id}

### Delete Employee
DELETE /api/Employee/{id}

## Learning Objectives

- Build RESTful APIs using ASP.NET Core
- Implement CRUD Operations
- Use Entity Framework Core with SQL Server
- Apply Data Annotation Validations
- Configure Entities using Fluent API
- Document APIs using Swagger
- Test APIs through Swagger UI
